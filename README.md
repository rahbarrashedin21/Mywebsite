# StudySpark (Study Tips Platform)

Team: Rashedin Rahbar & Gethmin

## How to run (XAMPP)
1. Copy this folder into: `C:\xampp\htdocs\study-platform`
2. Start Apache + MySQL in XAMPP
3. Open phpMyAdmin and run: `sql/setup.sql`
4. Visit: `http://localhost/study-platform/`

## Pages
- Home: pages/index.php (carousel + video + brand intro)
- Resources: pages/resources.php (Page 2 requirements + DB CRUD manager inside)
- Planner: pages/planner.php (JS calculator)
- Study Tips: pages/tips.php (interactive accordion)
- Account: pages/account.php (sign-in/sign-up with JS validation)
- About: pages/about.php (accessibility + inclusivity + acknowledgement)
